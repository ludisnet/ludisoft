Pour ouvrir le fichier d'exemple dans l'Explorateur de solutions

  1. Cliquez sur Télécharger l'exemple.

    Le message Téléchargement de fichier s'affiche.

  2. Cliquez sur Ouvrir, puis dans la colonne de gauche de la fenêtre du dossier du zip, cliquez sur Extraire tous les fichiers.

    L'Assistant Extraction s'ouvre.

  3. Cliquez sur Suivant. Vous pouvez modifier le répertoire dans lequel les fichiers seront extraits, puis cliquer de nouveau sur Suivant.

    Assurez-vous que la case à cocher Afficher les fichiers extraits est activée, et cliquez sur Terminer.

  4. Double-cliquez sur le fichier .sln de l'exemple.

    La solution exemple est affichée dans l'Explorateur de solutions. Vous pouvez obtenir un avertissement de sécurité indiquant que l'emplacement de la solution n'est pas fiable. Cliquez sur OK pour continuer.

Générer et exécuter l'exemple Bibliothèques dans Visual Studio

  1. Dans l'Explorateur de solutions, cliquez avec le bouton droit sur le projet FunctionTest, puis cliquez sur Définir comme projet de démarrage.

  2. Dans l'Explorateur de solutions, cliquez avec le bouton droit sur le projet FunctionTest, puis cliquez sur Propriétés.

  3. Ouvrez le dossier Propriétés de configuration, puis cliquez sur Déboguer.

  4. Dans la propriété Arguments de la ligne de commande, entrez 3 5 10.

  5. Cliquez sur OK.

  6. Dans le menu Déboguer, cliquez sur Exécuter sans débogage. La bibliothèque est générée automatiquement dans Functions et le programme est exécuté.

Pour générer et exécuter l'exemple Bibliothèques à partir de la ligne de commande

  1. Utilisez la commande Changer de répertoire pour accéder au répertoire Functions.

  2. Tapez le texte suivant :

     Copier le code
    csc /target:library /out:Functions.dll Factorial.cs DigitCounter.cs

  3. Utilisez la commande Changer de répertoire pour accéder au répertoire FunctionTest.

  4. Tapez le texte suivant :

     Copier le code
    copy ..\Functions\Functions.dll .
csc /out:FunctionTest.exe /R:Functions.DLL FunctionClient.cs
